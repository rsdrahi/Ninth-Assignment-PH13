## Project Name
Doctor Appointment Manager (DocAppoint)

## Purpose
This is a Doc Appoint web page. User can search a doctor and appoint for booking next serial. User can easily book an appointment anytime also update data. In the other hand anytime medial Emergency anyone call for Ambulance.

## Live Site link
- https://doc-appoint-beta.vercel.app

## Key Features
- Google Authentication Implement for secure
- User Friendly Interface
- Responsive for All Device
- Book appointments instantly without calling
- Browse doctors by specialty
- Quickly find suitable doctors
- Cancel or reschedule
- Track appointment history
